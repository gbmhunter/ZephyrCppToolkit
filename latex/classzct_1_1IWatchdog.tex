\doxysubsection{Detailed Description}
Interface for a hardware watchdog peripheral (not the software watchdog, this does not need to be mocked). 

This class provides a C++ abstraction layer over Zephyr\textquotesingle{}s watchdog API. It follows the same patterns as other Zephyr\+Cpp\+Toolkit peripheral interfaces.
