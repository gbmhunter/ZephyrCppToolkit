\doxysubsection{Detailed Description}
\doxylink{classzct_1_1Mutex}{Mutex} is a C++ wrapper around a Zephyr mutex.

The recommended way to lock a mutex is to use the \doxylink{classzct_1_1MutexLockGuard}{Mutex\+Lock\+Guard} class.
